# `orang help`

Displays help\.

## Synopsis

```
orang help [<COMMAND>]
[-m|--manual]
[-v|--values]
```

## Arguments

**`<COMMAND>`**

Command name\.

## Options

**`[-m|--manual]`**

Display full manual\.

**`[-v|--values]`**

Display list of allowed values\.
